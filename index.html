Manual para subir tu sitio web a x10Hosting

1. Crear una cuenta en x10hosting

Ve a x10hosting.com, regístrate y elige el plan gratuito.

2. Acceder al panel de control

Desde tu dashboard, entra a "Control Panel" para acceder al administrador de archivos.

3. Subir tus archivos

  • Usa el File Manager para subir tus archivos a la carpeta public_html.
  • O usa un cliente FTP como FileZilla con tus datos de acceso.

4. Verifica tu sitio

Ingresa la URL que elegiste al registrarte. Ejemplo: https://tunombre.x10.mx

5. Consejos adicionales

No olvides tener un archivo index.html como página de inicio.
